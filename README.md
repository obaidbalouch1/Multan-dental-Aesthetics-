# Multan Dental Aesthetic

Official website of **Multan Dental Aesthetic** — a modern dental clinic in
Multan, Pakistan, led by **Dr. Alizay Khan** (BDS, RDS).

## About the clinic

Multan Dental Aesthetic offers personal, one-on-one dental care — every
patient is treated by Dr. Alizay Khan herself, from the first consultation
to the final result.

### Treatments

- Veneers & Smile Design
- Teeth Whitening
- Orthodontics & Braces
- Dental Implants
- Root Canal Therapy
- Pediatric (Children's) Dentistry
